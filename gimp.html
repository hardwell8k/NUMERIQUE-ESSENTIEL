GIMP (GNU Image Manipulation Program) is an advanced open-source image editor used for photo retouching, composition, and authoring.


Visit the official website: gimp.org

Key Features

  • Powerful image retouching and editing tools
  • Supports layers and masks
  • Extensible with plugins
  • Cross-platform (Windows, Linux, Mac)
